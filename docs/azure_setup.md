# Azure Setup
Everything in my azure setup for this project is contained within my sunuxu resource group.

I have 2 function apps:
    - sunuxu-production-functions
    - sunuxu-test-functions

1 SQL server:
    - sunuxu

2 SQL databases:
    - test_sql
    - production-sql

1 storage account:
    - sunuxustorage

2 blob containers:
    - test-container
    - production-container
